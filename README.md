# Project Overview
Wheat diseases can severely affect agricultural productivity if not detected early. This project uses **Convolutional Neural Networks (CNN)** to classify wheat leaf images into healthy or diseased categories and provides predictions through a **Flask-based web application**.
# Key Features
- Image-based wheat disease classification  
- CNN model for accurate prediction  
- Flask web application interface  
- User-friendly design  
- Supports early detection and intervention
